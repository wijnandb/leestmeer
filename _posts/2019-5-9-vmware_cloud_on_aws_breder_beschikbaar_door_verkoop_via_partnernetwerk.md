Amazon Web Services (AWS) en VMware maken hun gezamenlijk ontwikkelde VMware Cloud on AWS-dienst breder beschikbaar door het open te stellen tot het partnernetwerk van Amazon. Ook wordt de dienst beschikbaar gemaakt via het eigen verkoopteam van AWS. Klanten kunnen VMware Cloud on AWS vanaf nu direct via AWS en bedrijven in het AWS Partner [&#8230;]
