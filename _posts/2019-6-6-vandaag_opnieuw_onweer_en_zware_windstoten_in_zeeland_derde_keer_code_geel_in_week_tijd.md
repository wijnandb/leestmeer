VLISSINGEN - Het KNMI heeft voor vanmiddag code geel afgekondigd voor Zeeland. Het gaat flink onweren en er zijn zeer zware windstoten mogelijk. Ook in de rest van het land is voor vanmiddag code geel afgekondigd.
