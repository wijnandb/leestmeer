Microsoft heeft een vroege previewversie van zijn vernieuwde Windows Terminal vrijgegeven via de Microsoft Store. De app is gratis te downloaden en installeren op Windows 10. Microsoft onthulde de nieuwe Windows Terminal tijdens zijn afgelopen Build-conferentie vorige maand. De code om de terminal zelf te compileren was al langer beschikbaar via GitHub.
