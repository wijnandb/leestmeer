Met zo'n 500 kilometer aan file op de Nederlandse snelwegen is er vrijdag sprake van een zeer zware avondspits, meldt Rijkswaterstaat Verkeersinformatie. Rond 17.15 uur werd volgens de ANWB in totaal 852 kilometer file gemeten.
