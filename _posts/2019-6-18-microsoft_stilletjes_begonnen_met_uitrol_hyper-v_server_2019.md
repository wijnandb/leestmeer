Microsoft is stilletjes begonnen met het uitrollen van zijn Hyper-V Server 2019. De uitrol begon afgelopen zaterdag, nadat topmensen van het bedrijf maandenlang beloofden dat de Hyper-V Server bijna algemeen beschikbaar zou zijn, meldt ZDNet.  De beschikbaarheid van Hyper-V Server 2019 werd kort aangekondigd op de Windows Server Insider-blog van Microsoft.
