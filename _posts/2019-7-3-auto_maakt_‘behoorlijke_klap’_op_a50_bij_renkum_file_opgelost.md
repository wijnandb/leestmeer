RENKUM - Door een ongeval is woensdagmiddag een file ontstaan op de A50 tussen Renkum en het knooppunt Grijsoord. De rechterrijstrook is een tijdlang afgesloten geweest, meldt Rijkswaterstaat. Tegen 14.30 uur was de file opgelost.
