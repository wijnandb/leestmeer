De Apache Software Foundation heeft nieuwe versies van Lucene en Solr uitgebracht met 8.1.1 als versienummer. Lucene kan worden ingezet als onderdeel van een zoekmachine. Het kan namelijk volledige tekstindexering uitvoeren. Solr kan vervolgens worden ingezet als een zoekmachine waarbij het op de achtergrond gebruikmaakt van Lucene en draait als een Java Servlet binnen bijvoorbeeld Tomcat. De aankondigingen van deze uitgaves zien er als volgt uit: Apache Lucene 8.1.1 available This release contains no change over 8.1.0.
