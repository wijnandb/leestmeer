De onweersbuien van vrijdag lijken mee te vallen. Voor de provincies Utrecht, Limburg, Noord-Brabant, Zeeland en Zuid-Holland geldt helemaal geen weeralarm meer, terwijl het voor Flevoland is bijgesteld naar code geel.
