Microsoft Azure kent meer Linux-gebruikers dan Windows Server-gebruikers. In 2017 was 40 procent van de Azure virtual machines Linux-gebaseerd en vorig jaar liep het percentage al op naar 50 procent. Sasha Levin, Microsoft Linux kernel-ontwikkelaar, heeft om die reden Microsoft verzocht toe te treden tot de Linux-beveiligingslijst.
