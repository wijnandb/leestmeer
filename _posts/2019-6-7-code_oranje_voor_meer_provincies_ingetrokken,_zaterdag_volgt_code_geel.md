De onweersbuien van vrijdag lijken mee te gaan vallen. Voor de provincies Utrecht, Limburg, Noord-Brabant, Zeeland en Zuid-Holland geldt helemaal geen weeralarm meer, terwijl voor Flevoland het is bijgesteld naar code geel.
