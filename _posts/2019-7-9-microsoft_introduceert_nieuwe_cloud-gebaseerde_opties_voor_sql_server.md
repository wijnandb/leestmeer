Microsoft heeft nieuwe cloud-hosted versies van zijn SQL Server-databasedienst aangekondigd. De ondersteuning voor twee oudere on premise-varianten is vandaag stopgezet. De nieuwe versies vervangen SQL Server 2008 en 2008R2, en zijn gehost op het cloud-platform Azure, schrijft Silicon Angle.
