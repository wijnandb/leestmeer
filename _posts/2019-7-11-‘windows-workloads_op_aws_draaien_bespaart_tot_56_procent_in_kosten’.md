Amazon Web Services (AWS) ziet natuurlijk graag dat zijn klanten Windows-workloads op het platform draaien. Om de voordelen daarvan te benadrukken, deelde vicepresident Sandy Carter resultaten van een IDC-onderzoek op LinkedIn. Volgens dat onderzoek kunnen organisaties die Windows-workloads als Windows Server en Microsoft SQL Server draaien, hun kosten in vijf jaar verminderen met maximaal 56 [&#8230;]
