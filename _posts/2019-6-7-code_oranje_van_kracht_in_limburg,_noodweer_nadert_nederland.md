In Limburg is om 17.00 uur als eerste provincie code oranje van kracht gegaan. De rest van de provincies volgen tussen 18.00 en 20.00 uur, voor Zeeland, Noord-Brabant, Zuid- en Noord-Holland is code geel afgegeven.
