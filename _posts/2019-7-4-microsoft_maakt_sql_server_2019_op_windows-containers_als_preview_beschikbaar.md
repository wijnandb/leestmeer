Microsoft heeft zijn SQL Server 2019 beschikbaar gemaakt als Windows-gebaseerde container images, schrijft The Register. Het gaat om een preview versie in het Early Adopter Program. Eerder waren alleen Linux-gebaseerde container images beschikbaar. De SQL Server 2019 werd vorig jaar al onthuld.
