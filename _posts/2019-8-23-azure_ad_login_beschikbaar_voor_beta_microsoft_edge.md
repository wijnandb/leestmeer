Op de beta-channels voor Microsoft Edge is het inloggen met Azure AD-accounts vanaf nu mogelijk. Dit heeft Microsoft bekendgemaakt in een blogbericht. De functie is een antwoord op feedback die het bedrijf kreeg van beta-gebruikers.
