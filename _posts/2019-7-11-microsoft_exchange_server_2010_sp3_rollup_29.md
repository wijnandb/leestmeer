Microsoft heeft alweer de negenentwintigste Rollup voor het derde servicepack van Exchange Server 2010 vrijgegeven. Service Pack 3 werd uitgegeven in februari 2013. De mainstreamondersteuning vanuit Microsoft eindigde in januari 2015, maar de verlengde ondersteuning loopt nog tot januari 2020. In deze verlengde periode worden alleen beveiligings- en tijdzone-updates doorgevoerd. Het bijbehorende KB-artikel met nummer 4509410 meldt het volgende: Update Rollup 29 voor Exchange Server 2010 SP3 Dit updatepakket is een beveiligingsupdate waarmee een beveiligingsadvies in Microsoft Exchange.
